1. 한 줄 정의

전략정의역할
composite_a BTC_EMA_bear OR ETH_30d_drawdown 기존 롱 진입을 막는 베어/위험 레짐 필터
composite_h BTC_EMA_bear OR ETH_30d_drawdown OR ETH_1d_ADX(14) > 28 composite_a 위에 2023형 과열 추세 차단 규칙 추가

2. 상속 구조

D11 bull_highvol_only
  -> D12 btc_filter
  -> D14 composite_a
  -> D16 composite_h
`composite_a`는 D12의 BTC bear filter 위에 ETH drawdown 레이어를 얹은 버전. `composite_h`는 `composite_a` 위에 F13 detector(ETH 1d ADX high)를 추가한 버전.

3. 구성 요소

3.1 C1 = BTC_EMA_bear

  • 정의: BTC 1d EMA20 < EMA60
  • 의미: BTC 일봉 기준 베어 레짐이면 롱 진입 차단
  • 코드: btc_filter 전략

3.2 C2 = ETH_30d_drawdown

  • 정의: ETH 5m 현재가가 직전 30일 rolling max 대비 -12% 아래
  • 의미: ETH 자체가 꽤 크게 꺾였으면 롱 진입 차단
  • 코드: composite_detector.py

3.3 C5 = ETH_1d_ADX_high

  • 정의: ETH 일봉 ADX(14) > 28
  • 의미: 일봉 추세가 너무 강한 구간은 5분봉 롱에 불리하다고 보고 차단
  • 배경: 2023 손실 구간이 오히려 “chop”보다 “과열된 강한 추세 후 꼬임”에 가까웠다는 D16 결과

4. 코드에서 실제로 어떻게 동작하나

4.1 composite_a

class ..._composite_a(_CompositeBase):
    def _composite_bear(self, dataframe):
        return dataframe["c2_eth_drawdown_bear"]
주의점: 클래스 안에서는 c2만 보이지만, 부모 클래스가 이미 btc_ema_bear를 먼저 막고 있음. 그래서 실제 의미는 D12 BTC filter + C2. 문서 표기상 D14.A = BTC_EMA_bear OR ETH_drawdown_30d.

4.2 composite_h

class ..._composite_h(_CompositeBaseF):
    eth_1d_adx_threshold = 28.0
    def _composite_bear(self, dataframe):
        return dd | adx_high
실제 의미: 부모의 BTC bear 차단 + 현재 클래스의 drawdown OR adx_high.
즉 최종 논리는 BTC_EMA_bear OR ETH_drawdown_30d OR ETH_1d_ADX(14) > 28.

5. 왜 composite_a가 먼저 나왔나

문제기존 상태A의 해결 시도
2026 bear catastrophic D11 long이 크게 손실 BTC bear filter가 강하게 차단
2023 ETH-only 약세 BTC만 보면 bull로 오판 ETH 30일 drawdown 조건으로 ETH decoupling 보완
핵심 아이디어: BTC만 보면 안 된다. BTC는 괜찮아 보여도 ETH 자체가 30일 고점 대비 충분히 밀렸으면 long 금지.

5.1 composite_a의 장점

  • D11, D12보다 5년 기준 가장 나은 baseline으로 올라옴
  • 2024, 2025는 잘 살림
  • 단순 구조라 해석 쉬움

5.2 composite_a의 한계

  • 2023 catastrophic 미해결
  • actual 기준 5년 누적은 여전히 음수
  • 2026은 아예 거래가 0건인 구간 발생

6. 왜 composite_h가 나왔나

D16의 핵심 발견: 2023 손실은 단순 chop보다 ETH 일봉 추세가 너무 강한 상태와 더 관련 있었음.
항목2023 losers2024 winners해석
F13 ETH 1d ADX(14) median 33 median 18 2023 진입 시점이 더 강한 일봉 추세 구간
  • composite_f: F13 > 30
  • composite_g: F13 > 33
  • composite_h: F13 > 28
`h`는 P0 후속 분석에서 threshold 28이 가장 강하다고 판단해 추가된 lower-threshold 변형.

6.1 composite_h에서 중요했던 수정

  • startup_candle_count를 2400 -> 7499로 증가
  • 이유: 1d ADX(14)는 초기 warmup이 부족하면 초반 구간이 전부 False로 죽을 수 있었음
  • 결과: post-fix final number 기준 +210.16%
